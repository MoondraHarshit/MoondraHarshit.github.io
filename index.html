About Me

Project 01

MPC + Residual Reinforcement Learning for Quadrupedal Robots

A hybrid locomotion architecture combining convex model predictive control with a Soft Actor-Critic residual policy for the Unitree A1 quadruped in MuJoCo. The learned policy augments MPC with foothold corrections and joint-torque residuals on rough terrain.

MPC Residual RL SAC MuJoCo Unitree A1 Quadruped Control
Key result The residual policy reduced body-attitude excursions in several rough-terrain evaluations, while revealing a trade-off between stabilization and forward-speed tracking.

Project 02

MIST: Latent Fault Adaptation for Multi-Drone Payload Transport

A multi-agent reinforcement learning framework that combines a GRU-based context encoder with MAPPO to infer hidden actuator faults during cooperative UAV payload transport.

MAPPO Multi-Agent RL GRU Fault Adaptation PyBullet UAVs
Key result MIST achieved performance comparable to vanilla MAPPO while substantially reducing per-scenario variance across several actuator-fault configurations.
GitHub Read Report
Project 03

Benchmarking GAIL Against Imitation-Learning Baselines

A PyTorch reproduction and benchmarking study comparing Generative Adversarial Imitation Learning, DAgger, Behavioral Cloning, and PPO under clean and noisy expert demonstrations.

GAIL DAgger Behavioral Cloning PPO PyTorch Gymnasium
Key result GAIL outperformed Behavioral Cloning and PPO on noisy CartPole demonstrations, while Behavioral Cloning remained more stable than GAIL on Acrobot, showing that adversarial imitation performance is strongly environment-dependent.
