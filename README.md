# Hi, I’m Harshit Moondra

I am an M.S. Engineering student specializing in Robotics at Purdue University and a Graduate Research Assistant in the SCALE Robotics Lab.

My work focuses on robot learning, model predictive control, imitation learning, multi-agent reinforcement learning, mobile manipulation, and autonomous systems.

## Current research

* Vision-guided mobile manipulation using ROS 2, MoveIt, and OpenCV
* Action Chunking Transformers for robotic manipulation
* Safe and uncertainty-aware model predictive control
* Multi-agent reinforcement learning for cooperative robotic systems
* Koopman-based representations for robot learning and control

## Featured projects

### MPC + Residual Reinforcement Learning for Quadrupedal Robots

A hybrid control architecture combining convex MPC with a Soft Actor-Critic residual policy for Unitree A1 locomotion in MuJoCo. The learned policy generates foothold corrections and joint-torque residuals to improve stability over rough terrain.

### Benchmarking GAIL Against Imitation-Learning Baselines

A PyTorch benchmarking framework comparing GAIL, DAgger, behavioral cloning, and PPO under noisy expert demonstrations and covariate shift.
